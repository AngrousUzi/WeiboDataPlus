下载文件后，pip install方法安装好csv,time,random,requests,re,bs4,urllib,pandas,pyecharts几个包即可

根据自己的需要修改names，注意，names是唯一需要修改的部分，只需要把你需要获得信息的微博用户的微博昵称加引号完整写入列表中即可，特别注意，其中每个成员的昵称不能有空格，不能有缺省，昵称是确认信息采集正确的唯一依据
